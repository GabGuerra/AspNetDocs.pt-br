Este tutorial ensina a usar o ASP.NET Core MVC e o Entity Framework Core com controladores e exibições. O Razor Pages é uma nova alternativa no ASP.NET Core 2.0, um modelo de programação baseado em páginas que torna a criação da interface do usuário da Web mais fácil e produtiva. É recomendável tentar o tutorial das [Páginas Razor](xref:data/ef-rp/intro) na versão do MVC. O tutorial do Razor Pages:

* É mais fácil de acompanhar.
* Fornece mais práticas recomendadas do EF Core.
* Usa consultas mais eficientes.
* É mais atual com a API mais recente.
* Aborda mais recursos.
