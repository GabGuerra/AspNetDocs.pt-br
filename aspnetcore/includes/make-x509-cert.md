No Windows, é possível criar certificados autoassinados usando o [cmdlet New-SelfSignedCertificate do PowerShell](/powershell/module/pkiclient/new-selfsignedcertificate?view=win10-ps). Para ver um exemplo sem suporte, confira [UpdateIISExpressSSLForChrome.ps1](https://github.com/aspnet/Docs/tree/master/aspnetcore/includes/make-x509-cert/UpdateIISExpressSSLForChrome.ps1).

Nas plataformas macOS, Linux e Windows, é possível criar certificados usando o [OpenSSL](https://www.openssl.org/).
