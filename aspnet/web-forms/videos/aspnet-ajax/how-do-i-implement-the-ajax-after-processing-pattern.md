# <a name="how-do-i-implement-the-ajax-after-processing-pattern"></a>[Como fazer:] Implementar o AJAX após o processamento padrão?

por [Joe Stagner](https://github.com/JoeStagner)

A estrutura ASP.NET AJAX permite que seu código do lado do servidor conectar-se no e responder a vários eventos do lado do cliente. Neste vídeo, saiba como implementar o após processamento padrão, que permite que o código do lado do servidor executar a lógica de conclusão depois que o usuário termina com a página da Web.
