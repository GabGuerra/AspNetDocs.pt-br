<span data-ttu-id="111b4-420">Existem oportunidades para violações de simultaneidade em todos os aplicativos que permite que vários usuários simultâneos para atualizar ou excluir dados.</span><span <span data-ttu-id="111b4-421">Se tais violações não são levados em consideração, quando dois usuários atualiza simultaneamente os mesmos dados quem obtém a última gravação "vence", substituir o outro
